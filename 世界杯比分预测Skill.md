## 快速使用

1. 触发 `world-cup-predict` Skill
2. 按北京时间联网（中文源）→ 更新 schedule / standings / **players.json**
3. 同组按赛序(K)预测未来 **2 天** → 写入 predictions.json
4. `python3 scripts/inject_dashboard.py`
5. 打开 `dashboard/index.html`

## 推理执行指令（AI必须按以下步骤执行）

### 步骤1：数据清洗与基准建立
1. 提取两队近10场正式比赛（优先世界杯、洲际杯、预选赛）的场均xG、xGA（预期失球）、PPDA、定位球进球占比
2. 计算两队Elo评分差值，映射为胜率基准（Elo差100分≈64%胜率）
3. 标注数据可信度：⚠️ 友谊赛数据权重降低50%，近6个月数据权重提升30%

### 步骤2：战意与策略博弈分析
基于出线形势，判断以下情景：
- **情景A（生死战）**：必须赢球/净胜球，预测进攻倾向提升，防守风险增加
- **情景B（打平即出线）**：预测保守战术，进球数预期下调
- **情景C（已出线/已淘汰）**：轮换可能性，战意不确定性增加
- **情景D（淘汰赛路径选择）**：是否存在战略性控分以避免强敌

**输出**：两队预期阵型变化（如：4-3-3→5-4-1）、攻防节奏预测

### 步骤3：战术克制矩阵
构建5维对抗模型：

| 维度 | 球队A优势 | 球队B优势 | 均衡 | 关键依据 |
|------|-----------|-----------|------|----------|
| 控球vs反击 | | | | |
| 边路vs中路 | | | | |
| 高位逼抢vs后场出球 | | | | |
| 定位球攻防 | | | | |
| 体能vs经验 | | | | |

**输出**：标注"决定性维度"（即最可能导致比分差异的1-2个维度）

### 步骤4：外部变量修正
- **气候/海拔**：对欧洲球队/南美球队/非洲球队的体能影响系数
- **裁判风格**：主裁判场均出牌数、点球倾向、比赛流畅度容忍度
- **舆论压力**：国内媒体评分、球迷期待值、历史包袱（如"复仇战"）
- **赛程密度**：休息时间差异（每少休息1天，体能系数-3%）

### 步骤5：概率化预测输出
必须输出以下结构化结果：

#### 5.1 比分概率分布（Top 8）
```
1-0: __%  |  依据：____
1-1: __%  |  依据：____
2-1: __%  |  依据：____
0-0: __%  |  依据：____
2-0: __%  |  依据：____
1-2: __%  |  依据：____
0-1: __%  |  依据：____
2-2: __%  |  依据：____
其他比分合计：__%
```

#### 5.2 衍生预测
- **胜平负**：主胜__% / 平局__% / 客胜__%
- **总进球**：0-1球__% / 2-3球__% / 4+球__%
- **双方进球（BTTS）**：是__% / 否__%
- **半场/全场**：____概率最高
- **角球数**：大__角__% / 小__角__%
- **黄牌数**：大__张__% / 小__张__%

#### 5.3 置信度评级
- ⭐⭐⭐⭐⭐（>85%）：数据充分，模型拟合度高
- ⭐⭐⭐⭐☆（70-85%）：1个关键变量存在不确定性
- ⭐⭐⭐☆☆（55-70%）：2个以上变量不确定或数据缺失
- ⭐⭐☆☆☆（<55%）：信息不足，预测仅供参考

### 步骤6：反事实与动态修正
**假设触发条件**：
- 若[球队A]第30分钟红牌：比分预测调整为____，依据____
- 若半场0-0且[球队A]必须赢球：下半场预测调整为____
- 若[球队B]先进球：[球队A]扳平概率__%，逆转概率__%

### 步骤7：模型自检
回答以下问题：
1. "我的预测与主流赔率（Bet365/威廉希尔）的差异点在哪里？"
2. "如果我的预测错了，最可能的原因是什么？"
3. "哪些信息缺失会导致我推翻当前结论？"

5. 如果数据不足，明确标注[数据缺失：____]，并降低该维度权重而非猜测

### 比赛信息
- **比赛**：德国 vs 日本
- **轮次**：2026世界杯小组赛E组第3轮
- **时间**：2026年6月22日 19:00（当地时间）
- **地点**：洛杉矶SoFi体育场，气温28℃，湿度45%

### 关键博弈点
- 德国必须赢球，且需净胜球超过日本（目前-4 vs +1，至少赢2球才安全）
- 日本打平即出线，但输球可能被淘汰（若哥斯达黎加赢西班牙）
- 西班牙若输球且德国大比分赢球，可能掉到第二

### 最终预测总结

| 项目 | 预测结果 |
|------|----------|
| **最可能比分** | **2-1 德国胜**（概率18.5%） |
| **纯实力比分** | 2-0 |
| **博弈修正比分** | 1-1 |
| **主导因素** | [均衡] |

## 四、博弈情景扩展（A–I）

详见 `.cursor/skills/world-cup-predict/game-theory.md`

| 情景 | 说明 |
|------|------|
| E 战略性控分 | 强队已确保出线，可能接受平局（如巴西 vs 摩洛哥 → 预测 1-1 而非 3-0） |
| F 协同博弈 | 同轮多场同时开球，交叉影响战意 |

| I 轮换保存 | 为淘汰赛留力 |

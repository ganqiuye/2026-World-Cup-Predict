# 博弈情景库（A–L）

## 权重公式（v2）

完整推导见 [weights-algorithm.md](weights-algorithm.md)。

```
P(h,a) ∝ exp( 0.32×S_str + 0.24×S_gt + 0.14×S_seq + 0.12×S_path + 0.10×S_pl + 0.08×S_ext )
```

| 维度 | 权重 | 情景 |
|------|------|------|
| 纯实力 | 0.32 | — |
| 博弈修正 | 0.24 | A–I |
| 同组赛序 | 0.14 | **K** |
| 淘汰赛路径 | 0.12 | J |
| 球员个人 | 0.10 | **L** |
| 战术/外部 | 0.08 | — |

每场比赛必须标注：`[博弈主导]` / `[实力主导]` / `[均衡]`

必须输出：
- **纯实力最可能比分**（忽略出线形势）
- **博弈修正后最可能比分**
- **weight_breakdown** 与 **weight_contributions**
- **修正原因**（1–3 句）

## 情景判定表

| 代号 | 名称 | 触发条件 | 预测影响 |
|------|------|----------|----------|
| **A** | 生死战 | 必须赢球或特定净胜球才能出线/晋级 | 进攻↑，比分方差↑，黄牌↑ |
| **B** | 打平即出线 | 平局确保晋级（含相互战绩占优） | 保守战术，平局概率 +8~15% |
| **C** | 已出线/已淘汰 | 名次已定或出局 | 轮换，实力系数 ×0.75~0.85 |
| **D** | 淘汰赛路径选择 | 某名次可避开淘汰赛强敌 | 可能战略性控分 |
| **E** | 战略性控分 | 赢球反而遇强敌 / 小组第二路径更优 | 强队接受平局或小负 |
| **F** | 协同博弈 | 同轮多场**同时**开球，A 场结果影响 B 场 | 交叉引用同轮比赛，动态调整 |
| **G** | 净胜球算计 | 需赢 N 球但不必大胜 | 领先后收手，大比分概率↓ |
| **H** | 默契球风险 | 两队打平均可携手出线 | 平局概率 +10~20% |
| **I** | 轮换保存 | 已锁定名次，为淘汰赛留力 | 冷门概率↑，xG 下调 20% |
| **J** | 淘汰赛路径胜率 | 小组第一 vs 第二导致 32 强对手强弱不同 | 已出线时倾向走胜率更高的路径，影响当前战意 ±5~15% |
| **K** | 同组赛序传导 | 同组本轮**先后**开赛，先赛结果改变后赛战意 | 见下方 K 专节 |
| **L** | 球员个人目标 | 金靴/最佳球员/纪录追逐 | 进攻核心 λ +5~8%；目标已达成则 −10% |
| **M** | 射门转化效率 | 高控球低射正/xGOT | λ −5~12% |
| **O** | 舆论反弹战意 | 首轮爆冷次轮正名 | λ +8~15%，禁用 G 收手 |
| **Q** | 最佳第三全局 | R3 跨组第三出线 | 与 F 联动 |

**L 强化（v2.1）：** must_win + 首轮未达标 + pressure≥0.7 + 弱旅 → L×1.5，关闭 G。  
**B+（v2.1）：** 同分且平局仍保前二、非末轮 → 平局 +12%（如英格兰 0-0 加纳）。

## 情景 K：同组赛序传导

**与 F 的区别：** F 是同轮**同时**开球（第三轮 12 场）；K 是**任意轮次**内先赛后赛的因果链。

### 触发条件

1. 同组存在 `status=finished` 且 `datetime_cst` 早于当前场的比赛（本轮或前轮）
2. 先赛结果改变当前场至少一支球队的 `scenario_code`

### 分析步骤

1. 按北京时间排序同组本轮场次
2. 用已赛比分重算临时积分榜
3. 列出「若当前场比分 X，则各队出线状态」
4. 对比先赛**前**与**后**的 scenario 变化

### 典型模式

| 先赛结果 | 后赛影响 |
|----------|----------|
| 榜首大胜提前出线 | 后赛榜首轮换（C+I+K），对手生死战（A+K） |
| 榜首意外丢分 | 后赛变三方混战，平局概率↓ |
| 第二名先赛赢球 | 后赛第三名必须赢（A+K） |
| 两弱队先赛平局 | 后赛强队可「控分」保体力（E+K） |

### 输出模板

```markdown
### 赛序传导（K）
- **先赛：** 阿根廷 2-0 奥地利（6/23 01:00 北京）→ 阿根廷锁定 J 组第一
- **本场影响：** 约旦/阿尔及利亚胜者积 3 分争第二，败者基本出局
- **λ 修正：** 约旦 +12% 进攻（A），阿尔及利亚 +10%（A）
```

## 情景 L：球员个人目标

数据来自 `data/players.json`（联网后更新射手榜）。

| 条件 | 修正 |
|------|------|
| 射手榜前 3 且球队需进球 | λ +5~8%，大比分权重 +0.10 |
| 最佳球员竞争者 + 关键战 | 核心出场时间↑，λ +5% |
| 个人纪录已达成 + 球队已出线 | λ −10%（如梅西 18 球后） |
| 末届世界杯老将 | 对弱旅进球意愿 +0.08 |

## 情景组合规则

- 多情景可叠加，修正系数累加但单维度上限 ±15%
- **F（协同博弈）** 在小组赛第三轮必须执行
- **K（赛序）** 在同组存在已赛本场前比赛时必须执行
- **L（球员）** 当 `players.json` 中该队有 pressure ≥ 0.65 的球员时必须执行
- 当 **E + I + K** 同时触发且对手较弱：最可能结果可能从「3-0」变为「1-1」或「2-0 收手」

## 48 队赛制特殊点

- 12 组前 2 + 8 个最佳第三名 → 第三名出线计算需跨组比较
- 小组赛第 3 轮 12 场同时开球 → **F 情景必分析**
- 第 1、2 轮同组先后开赛 → **K 情景必分析**
- 最佳第三名排名影响：积分 → 净胜球 → 进球 → 公平竞赛

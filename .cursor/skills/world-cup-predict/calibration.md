**球员 L 子规则修正：**
```
若 (must_win + 首轮未赢 + pressure≥0.7 + 对手弱旅)
```

## 4. 同组赛序 K：动态变化方案

### 4.1 当前问题

现实现为 **单点基准**：「若葡萄牙 3-0 → 哥伦比亚须赢」。  
实际葡萄牙 **5-0**，英格兰 **0-0**（非 2-1），后赛预测链断裂。

### 4.2 动态 K（v2.1 算法）

**Step 1 — 先赛结果集合**

对已赛先赛场 `m₁`，取 Top N 比分 `{ (h,a), p_i }`（来自先赛预测分布，或已赛实际 `{1.0}`）。

**Step 2 — 后赛边际概率**

```
P_post(h,a) = Σ_i  p_i × P(h,a | standings'(score_i))
```

**Step 3 — 实时触发**

| 事件 | 动作 |
|------|------|
| 先赛开球前 | 用 Step 2 边际预测后赛 |
| 先赛中场 | 用当前比分更新 `standings'`，重算后赛 |
| 先赛终场 | **强制**用 `actual_score` 重算后赛，更新 predictions.json |

**Step 4 — 跨轮次**

- R2 先赛后赛 → K
- **R3 12 场同开** → K 关闭，**F（协同博弈）** 权重升至 **0.22**，与 Q（最佳第三）联动

### 4.3 仪表盘实现建议

- `analysis.k_scenarios`: `[{ prior_score: "5-0", prob: 1.0, downstream_effect: "..." }]`
- 先赛结束后自动标记 `needs_repredict: true` 的后赛场次

## 6. 淘汰赛权重（v3）

淘汰赛与小组赛目标函数不同：**无默契球、无小组路径选择、每场即生死战**。

### 6.1 淘汰赛 v3 公式

```
P_knockout(h,a) ∝ exp(
  0.38×S_str + 0.22×S_gt + 0.06×S_fatigue + 0.10×S_ko_hist
  + 0.08×S_pl + 0.08×S_ext + 0.08×S_tactics
)
```

| 维度 | 权重 | 说明 |
|------|------|------|
| 纯实力 | **0.38** ↑ | 硬实力主导 |
| 博弈 | **0.22** ↓ | 仅 A（必须赢）+ 加时点球预期 |
| ~~赛序 K~~ | **0** | 淘汰赛无同组先后 |
| ~~路径 J~~ | **0** | 对阵已固定 |
| **体能/赛程 S_fatigue** | **0.06** | 新：距上场天数、120 分钟消耗 |
| **淘汰赛基因 S_ko_hist** | **0.10** | 新：世界杯/欧洲杯 KO 胜率、点球大战史 |
| 球员 L | **0.08** ↓ | 金靴仍有效，但团队优先 |
| 外部 | **0.08** | 不变 |
| **战术 S_tactics** | **0.08** | 新：防反 vs 控球在 KO 的放大效应 |

### 6.2 淘汰赛新增情景

| 代号 | 名称 | 影响 |
|------|------|------|
| **S** | 淘汰赛生死 | 每场 A，但平局概率基线 +15%（可能加时） |
| **T** | 点球大战预期 | 实力接近时 0-0、1-1 权重 ↑ |
| **U** | 消耗战 | 上半场 0-0 时下半场进球率 +8% |
